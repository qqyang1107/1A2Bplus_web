1A2B Guessing Game

Rules:

  • Guess a number consisting of unique characters.
  • The system will return feedback in the format XAYB:
    • X (A) - The number of digits that are correct and in the correct position.
    • Y (B) - The number of digits that are correct but in the wrong position.
  • Example: If the answer is "1234" and you guess "1243", the feedback will be 2A2B.
